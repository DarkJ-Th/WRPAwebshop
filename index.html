หมายเหตุสำคัญ

  • ราคาข้างต้นไม่รวมภาษีมูลค่าเพิ่ม (VAT 7%)
  • แพ็กเกจ Free อาจถูกจำกัดให้ Broadcast ได้ 300 ข้อความ/เดือน ตามข้อมูลปี 2024-2025
  • ค่าข้อความส่วนเกินจะแตกต่างกันตามแพ็กเกจที่เลือกใช้

⚠️ การเปลี่ยนแปลงและประเด็นสำคัญ

🔔

การปรับราคาและเงื่อนไขล่าสุด

  • ปี 2023-2024: แพ็กเกจ Free ลดโควต้าข้อความจาก 1,000 → 500 ข้อความ/เดือน

  • ราคาข้อความเกินโควตา: 0.06–0.10 บาท/ข้อความ (ขึ้นอยู่กับแพ็กเกจ)

1ดาวน์โหลดแอป LINE Official Account

A Android

  • เปิด Play Store
  • ค้นหา: LINE Official Account
  • กด ติดตั้ง

i iPhone

  • เปิด App Store
  • ค้นหา: LINE Official Account
  • กด Get

2สร้างบัญชี LINE OA

  • เปิดแอป LINE Official Account
  • กด Create account
  • ตั้งชื่อ OA และเลือกหมวดหมู่
  • กด Create

3เข้าเว็บ LINE OA Manager

  • ไปที่เว็บ https://manager.line.biz
  • ล็อกอินด้วยบัญชี LINE
  • เลือก OA ที่สร้างไว้

4เปิดใช้ Messaging API

⚠️ จุดที่ทำให้หลายคนงง! หลังจากเปิด API แล้ว การตั้งค่าจะย้ายไปที่ LINE Developers Console
  • ใน OA Manager เลือกเมนู Settings
  • เลือก Messaging API
  • กด Enable Messaging API
  • ระบบจะให้เลือก Provider
  • ถ้าไม่มี Provider ให้กด Create new Provider
  • กดตกลงจนจบกระบวนการ
📢 หลังจากเปิดสำเร็จ:
ระบบจะขึ้นแจ้งว่า "ให้ไปตั้งค่าที่ LINE Developers Console"
นี่คือปกติ เพราะหลังจากเปิด API แล้ว การตั้งค่าจะย้ายไปที่ https://developers.line.biz

5ไปตั้งค่าต่อใน LINE Developers Console

  • เปิดเว็บ https://developers.line.biz/console
  • ล็อกอินด้วยบัญชี LINE
  • เลือก Provider ที่สร้างไว้
  • เลือก Messaging API Channel ที่เชื่อมกับ OA
🎯 ข้อมูลสำคัญในหน้านี้:
• Channel Secret
• Channel Access Token
• Webhook settings
• Bot options

6เปิด "Allow bot to join chat" (สำคัญมาก!)

⚠️ สำคัญมาก: ถ้าไม่เปิด บอทจะไม่เห็นข้อความในกลุ่ม และไม่สามารถเข้าได้เลย
  • ทำใน แอป LINE OA (ไม่ใช่ใน Developers Console)
  • เปิดแอป LINE Official Account
  • เข้า OA ของคุณ
  • กดรูปฟันเฟือง Settings
  • เลือก Chats
  • เปิด Allow bot to join chats → ON

7เปิดใช้ Webhook แบบถูกต้อง

  • ทำใน LINE Developers Console
  • ไปที่หน้า Channel (Messaging API)
  • เลื่อนลงไปหัวข้อ Webhook settings
  • เปิดสวิตช์ Use webhook → ON
  • ช่อง Webhook URL → ยังไม่ใส่ (เราจะใส่ในขั้นตอนต่อไป)

8สร้าง Webhook URL โดยใช้ webhook.site

  • ไปที่เว็บ https://webhook.site
  • ระบบจะสร้าง URL ส่วนตัวให้ เช่น https://webhook.site/12345-abcdef
  • คัดลอก URL นั้นไว้

9ใส่ Webhook URL บน LINE Developers

  • กลับไปที่ https://developers.line.biz/console
  • เข้า Messaging API Channel
  • วาง URL ลงในช่อง Webhook URL
  • กด Verify
✅ ถ้าขึ้น "Success" แปลว่าพร้อมใช้งาน

10วิธีหา User ID ผ่าน Webhook.site

  • เปิดแอป LINE บนมือถือ
  • ทักข้อความใดก็ได้ไปที่ OA ของคุณ
  • ดูข้อมูลใน webhook.site จะมี JSON เด้งขึ้นมา
{ "events": [ { "type": "message", "source": { "type": "user", "userId": "Uxxxxxxxxxxxx" }, "message": { "type": "text", "text": "ข้อความที่ส่ง" } } ] }
🎯 ข้อมูลที่ต้องเก็บ:
• userId = ID ของผู้ใช้ที่ส่งข้อความ (ขึ้นต้นด้วย U)

11วิธีหา Group ID ผ่าน Webhook.site

  • สร้างกลุ่ม LINE
  • เชิญ OA เข้ากลุ่ม
  • พิมพ์ข้อความใดก็ได้ในกลุ่ม
{ "events": [ { "type": "message", "source": { "type": "group", "groupId": "Cxxxxxxxxxxxx", "userId": "Uxxxxxxxxxxxx" }, "message": { "type": "text", "text": "ข้อความที่ส่งในกลุ่ม" } } ] }
🎯 ข้อมูลที่ต้องเก็บ:
• groupId = ID ของกลุ่ม (ขึ้นต้นด้วย C)
• userId = ID ของผู้ส่งข้อความในกลุ่ม (ขึ้นต้นด้วย U)

12ข้อมูลสำคัญที่มักงง

📝 จุดสำคัญที่ต้องจำ:

1. หลังเปิด Messaging API ใน OA Manager → ต้องไปตั้งค่าส่วนใหญ่ใน LINE Developers Console

2. การรับข้อความ: ผู้ใช้ต้อง "ทัก OA ก่อน"

3. การรับข้อความจากกลุ่ม ต้อง:
  • เชิญ OA เข้ากลุ่ม
  • เปิด Allow bot to join chats
  • พิมพ์ข้อความอย่างน้อย 1 ครั้ง

📌 สรุปข้อมูลสำคัญที่ต้องเก็บ

🔑 จาก LINE Developers Console:
• Channel Secret
• Channel Access Token
🎯 จาก Webhook.site:
• userId (สำหรับส่งข้อความถึงผู้ใช้)
• groupId (สำหรับส่งข้อความถึงกลุ่ม)
ขั้นตอนโดยสรุป:
  • ดาวน์โหลดแอป → สร้าง OA → เปิด Messaging API
  • ย้ายไปตั้งค่าใน LINE Developers Console
  • เปิด Allow bot to join chats ในแอป LINE OA
  • ตั้งค่า Webhook โดยใช้ webhook.site
  • หา user ID / group ID ผ่านการส่งข้อความ
